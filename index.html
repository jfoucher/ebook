Ebook

Ebook app for Firefox OS

Download Download View Ebook on GitHub

Features

If you have epub files on your SD card, Ebook will load them on first run, preprocess them and they will be available to read within a few seconds or minutes, depending on the number of books you have.

The design is very nice, making it a pleasure to read. The font size can be changed by pinching in or out. It will be saved for all books.

Install Ebook on your Firefox OS device

Just go to the Firefox Marketplace and click the install button.
